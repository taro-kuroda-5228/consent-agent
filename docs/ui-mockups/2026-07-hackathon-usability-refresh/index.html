画面設計の方針
  • 医師: 30秒で開始、詳細は折りたたむ
  • 家族: 1画面1行動、動画→カード→質問
  • AI: 根拠限定・自律判定を見える化

本番適用前のデザイン専用モックです。デモ用合成症例のみ。実患者情報・秘密情報・外部送信は含みません。
